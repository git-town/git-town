Install GitTown

Git Town runs anywhere Git and Bash run.

Using Homebrew On Linux Manually
Install
brew install git-town
  • On Arch Linux, install git-town from the AUR
  • On Debian/Ubuntu based distros, download the deb file from the latest release here. Then, run dpkg -i /path/to/deb
    • clone the repo to your machine (into DIR)
    • add DIR/src to your $PATH
    • add DIR/man to your $MANPATH
    Update
    brew update
    brew upgrade git-town
    • Rerun the install instructions
    • run git pull in DIR
    Uninstall
    brew uninstall git-town
    • On Arch Linux run:
      pacman -R git-town
    • On Debian/Ubuntu run:
      apt-get remove git-town
    • remove DIR
    • remove DIR/src from your $PATH
    • remove DIR/man from your $MANPATH

    Install autocompletion

    • for Fish shell: git town install-fish-autocompletion
